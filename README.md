# HeroFigther

## Description
A FullStack Text-based RPG

## Tech Stacks
- HTML
- CSS,
- JavaScript,
- Koa
- PostgreSQL

  
## Getting Started

Run `npm install`.

To run the server:

```bash
npm run index.js
```
To run the client:

```bash
npm http-server -c-1
```

## Features
- Interactive buttons
- Able to move between doors
- Create a new player
- Add basic items at the inventory
- Tested server endpoints
- Components organized in modules
